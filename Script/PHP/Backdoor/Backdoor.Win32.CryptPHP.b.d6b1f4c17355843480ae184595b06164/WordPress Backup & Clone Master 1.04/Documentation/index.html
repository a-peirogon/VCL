Overview

WordPress Backup & Clone Master is an all-in-one solution for WordPress backup, restoration, cloning, and migration. The plugin helps you manage the above processes in a secure, easy, and reliable way on a scheduled or on-demand basis.

Features

- Create full WordPress backups, including content, settings, themes, and plugins
- Download backup to your computer in a ZIP or TAR file
- Schedule daily / weekly / monthly backups to be updated on your server
- Define how many backups should be stored on the server
- Auto upload backups to your Dropbox
- Send regular backups to your email address
- Restore any backup in one click
- Clone your whole site to a different domain
- Create a new database on a remote server directly from the plugin
- Clone your settings, themes, and plugins to a different WP site
- Migrate your site to a new host
- Works on any hosting environment
- Works with WordPress Network (Multisite)

Installation & Update

Installation

1. In your WordPress Admin area, go to the “Add New” section under “Plugins” tab.

2. Select the “Upload” option.

3. Browse to and select your “blogclone.zip” file and press the “Install Now” button.

4. After the plugin has been installed, click “Activate Plugin”.

5.  You will then see Backup & Clone Master in the list of your plugins. It will also create “Backup & Clone” tab on the left of your WordPress dashboard, where you can create and manage publishing projects. 

You might want to check the following manual featuring screenshots for the plugin installation process:

http://getbusinessblog.com/download/plugins/backup-clone/install-plugin.pdf

Update

When a new version of the Backup & Clone Master becomes available, you will see a notification in your WordPress dashboard -> Updates section. You are able to update the plugin directly from the WordPress dashboard without needing to re-install it.

Getting Started

1. In order to get started, choose if the plugin should generate your backups / clones as ZIP or TAR files. For this, click "Check Backup File Size" button and check if your backup file size does not exceed your host's PHP memory limit. If it does, use TAR archiving for backups and clones.

2. If you would like to create a full backup of your website, simply click "Backup / Clone" button at the bottom. It will create the backup of your whole site, including WordPress files, themes, plugins, content, media, and all database data. You can further download backup to your PC in the Backup Data section by clicking HERE link next to "Download Your Backup / Clone" field. You can also restore this backup any time by clicking "Restore This Backup" button below the link.

3. For auto backups of your site, check and setup the Backups on Server, Backup to Dropbox, and Backup to Email sections. After selecting the settings for auto backups, don't forget to click "Save Settings" button at the bottom of the page.

4. In order to restore your backup or clone, go to Backup Data section -> Upload & Run Backup File and browse for the required file. When the file is selected, you will see "Restore Backup" button. Click the button to restore the selected file. If the file size is larger than your maximum upload file size limit, contact your host to ask them to increase the limit.

5. If you would like to create a clone of your site to be installed on a different domain / server, select the required settings in the Backup & Clone Settings section. Then, replace the default data in the Blog Details and Database Details sections with the details of the site, where you would like to install the clone. When done, click "Backup / Clone" button. You can further download the clone to your PC in the Backup Data section by clicking HERE link next to "Download Your Backup / Clone" field. In order to install your clone, upload the generated file through FTP, or browse for the file in the Backup Data section and restore it on a new domain / server.

Backup & Restore

Keep Your Website Secure with Regular Backups

Backup & Clone Master plugin provides secure protection for WordPress sites and their data. You invest a great amount of time into your website, and it is essential to make sure your data is safe. With regular backups created, you don't need to worry about data loss in case of server injections, software upgrades, or site changes. It is also important to have an easy way to restore your backup without advanced knowledge of Cpanel and databases. Backup & Clone Master takes care of the backups, and lets you restore your site in one click, which will definitely bring you peace of mind and added safety.

Clone WordPress

Clone Your Website Easily to a New Domain

Cloning ability is especially valuable when you are running multiple WordPress sites. It could be a time consuming project to configure WordPress, install and customize theme, install and configure multiple plugins, which is usually the same setup for all of your websites. With the Backup & Clone Master plugin, you can simply copy the settings (including cusomized themes and configured plugins), or the whole site, to a different domain quickly and easily. This feature has a great potential for any web master, and is surely to save you tons of time when managing your WordPress sites.

Migrate Your Website Easily to a New Host

Moving your WordPress site to a new hosting server could be a headache for a non technical person as it involves new database creation, new WordPress installation, data migration, and configuration. With the Backup & Clone Master plugin, this has become a simple task, for which you don't need any advanced technical knowledge (the plugin will even create a database for you remotely!).
